differentiation
tag conformable different
hr & br ตัวแบ่ง element ใน HTML hr คือสร้างเส้นแนวนอน br คือ ขึ้นบรรทัดใหม่
ul & ol เรียงลำดับ ul คือ เรียงลำดับแบบสัญญาลักษณ์ ol คือเรียงลำดับแบบตัวเลข
select & checkbox & radio button คำสั่งการเลือกข้อมูล SELECT คือ คำสั่งที่ใช้ในการเลือกข้อมูลจากฐานข้อมูล checkbox คือ ฟอร์ม HTML ที่มีช่องทำเครื่องหมายที่เลือกไว้ล่วงหน้า radio button คือ กำหนดปุ่มตัวเลือก
div & span กล่องข้อความ div คือกล่องข้อความ อยู่บรรทัดถัดจาก element ก่อนหน้า span คือ กล่องข้อความที่อยู่ในบรรทัดเดียวกันกับelement ก่อนหน้า
script & style เป็นคำสั่งของHTML script คือ ประกอบด้วยคำสั่งการสคริปต์หรือชี้ไปที่ไฟล์สคริปต์ภายนอกผ่านแอตทริบิวต์ src style คือ ภายในองค์ประกอบ style คุณระบุว่าองค์ประกอบ HTML ควรแสดงผลอย่างไรในเบราว์เซอร์
